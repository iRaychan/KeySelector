## Replace your existing GitHub files

1. Extract this ZIP.
2. Open your `iraychan/KeySelector` repository.
3. Upload all extracted files to the repository root and replace the old files.
4. Commit the changes.
5. Wait for GitHub Pages to redeploy.
6. Open the site with `?v=313` once to bypass the old cache:
   `https://iraychan.github.io/KeySelector/?v=313`

If an installed phone app still shows the old version, remove the old home-screen app, open the URL in Safari/Chrome again, and reinstall it.
